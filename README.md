Test performance between test runners.

- Many test files (5000)
- 1 import per test
- 1 test per test file

# Usage

## node

```shell
cd node
node --test
```

## Jest

```shell
cd jest
npm i
npm exec jest
```

### Vitest

```shell
cd vitest
npm i
npm exec vitest
```

### bun

```shell
cd jest
npm i
bun test 
```
